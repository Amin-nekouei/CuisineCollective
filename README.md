# CuisineCollective

CuisineCollective is a dynamic recipe-sharing web application developed using PHP, JavaScript, HTML/CSS, and MySQL. The platform allows users to create, manage, and explore recipes through a responsive and user-friendly interface.

## Features

* User registration and authentication
* Recipe creation and management
* Responsive design for mobile and desktop
* Real-time form validation
* Secure database integration with MySQL

## Technologies

* PHP
* JavaScript
* HTML5 & CSS3
* MySQL

## Project Goal

This project was developed as a collaborative academic web application to strengthen practical skills in full-stack web development, database management, and responsive UI design.
